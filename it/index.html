FAQ

Domande frequenti

Dove sono archiviati i dati del mio portafoglio?

Le tue posizioni e i dati finanziari sono salvati sul tuo dispositivo. MyGoldFolio non carica valori del portafoglio, prezzi o dati personali. L’analisi anonima opzionale si attiva solo con consenso esplicito e può essere disattivata in qualsiasi momento in Impostazioni → Privacy. L’app continua a recuperare prezzi live da API pubbliche.

Quale versione Android è richiesta?

Android 10 (API 29) o successivo. L’app è sviluppata con React Native New Architecture ed è ottimizzata per Android moderno. Gli avvisi in background richiedono Android 12+ per una pianificazione affidabile tramite WorkManager.

Quali valute sono supportate?

8 valute supportate: EUR, USD, GBP, INR, AED, SAR, TRY ed EGP. I dati del portafoglio sono salvati internamente in EUR. La visualizzazione converte con tassi live (cache 1h). Il peso può essere mostrato in grammi, once troy o tola.

Cosa succede ai miei dati se disinstallo?

I tuoi dati verranno eliminati con l’app: non esiste un server di backup. Prima di disinstallare, usa la sezione Export e Backup in Impostazioni per salvare un backup JSON completo sul dispositivo. Potrai ripristinarlo dopo la reinstallazione.

L’app è gratuita?

Sì. MyGoldFolio include una base gratuita solida, senza annunci né abbonamento. Premium è uno sblocco opzionale una tantum Lifetime con limiti più alti e strumenti avanzati.

Quanto sono accurati i prezzi dell’oro?

I prezzi provengono da tre fonti integrate indipendenti, con passaggio automatico se una fallisce. I prezzi spot riflettono di norma dati quasi in tempo reale di Londra/New York durante l’orario di mercato. Possono esserci lievi ritardi e non sono adatti a decisioni di trading.
