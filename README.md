# MyGoldFolio Website

Marketing website for MyGoldFolio.

Live site: https://mygoldfolio.de

## Repo Purpose

This repository contains the static website for the MyGoldFolio product site,
including the landing page, gallery, help pages, changelog, privacy page, and
other public-facing product content.

## Stack

- Static HTML
- Tailwind via CDN
- GitHub Pages deployment

## Notes

- The public site domain is configured via `CNAME`.
- `version.js` is the single source of truth for the website version badge.
- The public changelog lives at `changelog/index.html`.
- The detailed internal changelog is kept locally in `docs/changelog-internalonly.html`
  and is not published.
